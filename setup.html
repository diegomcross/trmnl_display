Vault Verdict — Setup

Three steps, once per PC. Your Bungie password is never typed here — you log in on Bungie's own website and this page just saves the resulting connection. Re-running any step later is safe (use step 2 to log in again if your login ever expires).

… 1 · App keys

Found on this PC — nothing to do. (If you got the app from a friend with the .env file included, this is already done.)

The app needs its own (free) Bungie identity. One person does this once, then the resulting .env file can be shared with the app.

  1. Open bungie.net/en/Application and sign in → Create New App
  2. Application Name: anything (e.g. Vault Verdict) · Website: https://localhost
  3. OAuth Client Type: Confidential
  4. Redirect URL: https://127.0.0.1/auth
  5. Scope: tick every checkbox (read Destiny data, move and equip items)
  6. Origin Header: leave empty · agree to the terms · Save
  7. Copy the three values from that page into the boxes below.

… 2 · Log in with Bungie

Click the button — Bungie's own site opens in a new tab. Sign in there and click Authorize. You'll land on a page that says the site can't be reached — that's normal. Copy the entire address from the browser's address bar, come back here, paste it, and press Connect.

… 3 · DIM tag sync

Connects your keep/favorite/junk tags to the DIM cloud (so tags made here show in the DIM app and vice-versa). Fully automatic — one click.
